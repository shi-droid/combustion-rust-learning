# 認識燃燒與生鏽

這是一個依照附件照片需求整理並製作的互動式自然科學學習網頁，主題改為「認識燃燒與生鏽」，內容參考附件 PDF 第 2 單元。

## 生成後的清楚工程指令

請建立一個適合國小五年級學生使用的互動式網頁，主題為「認識燃燒與生鏽」。網頁需參考課本 PDF 中「空氣與燃燒的關係、氧氣和二氧化碳的特性、空氣與生鏽的關係」等重點，包含燃燒三要素、氧氣助燃、二氧化碳不助燃、滅火原理、鋼棉生鏽實驗、水與酸性水溶液對生鏽的影響、氧氣與生鏽的關係，以及防鏽方法。頁面需具備互動性、學習性與美觀畫面，讓學生透過切換條件、觀察模擬、完成任務與測驗來理解抽象概念。完成後以 HTML/CSS/JavaScript 建成可直接開啟的靜態網頁，並準備發布到 GitHub Pages。

## 檔案

- `index.html`：學習頁主結構
- `styles.css`：版面與視覺設計
- `app.js`：燃燒、生鏽、氣體實驗與測驗互動
- `server.js`：本機預覽用的小型伺服器
- `pdf_ocr.txt`：從 PDF 擷取出的 OCR 文字紀錄，供內容整理參考

## 使用方式

直接開啟 `index.html` 即可使用。若放到 GitHub Pages，根目錄保留這三個網頁檔案即可。

本機預覽也可以執行 `node server.js` 後開啟 `http://127.0.0.1:4173/`。
